# Digital LineCoding using PIC16F877A

This project implements digital line coding techniques using PIC16F877A microcontroller in Proteus simulation.

## Line Codes Implemented
- Manchester
- Differential Manchester
- HDB3
- MLT-3

## Features
- Mode selection using switches (RA0, RA1, RA2)
- Encoded output on RD0
- Timing control using software delay
- Simulated using Proteus

## Tools Used
- Micro c pro for pic 
- Proteus
- PIC16F877A
